Component Inputs

Input Type Description Defaults
[data] [WhiteboardElement] The whiteboard data []
[options] WhiteboardOptions component configuration object, properties described below null
[drawingEnabled] boolean Enable mouse/touch interactions true
[selectedTool] ToolsEnum The current selected tool ToolsEnum.BRUSH
[canvasWidth] number The width of whiteboard canvas 800
[canvasHeight] number The height of whiteboard canvas 600
[fullScreen] boolean if true change (canvasWidth, canvasHeight) to fit the parent container true
[strokeColor] string The default stroke color #000000
[backgroundColor] string The default background color #FFFFFF
[fill] number The default fill color #333333
[strokeWidth] number The default stroke width 2
[zoom] number Zoom level 1
[fontFamily] string The default font family sans-serif
[fontSize] number The default font size 24
[center] boolean Center the canvas in parent component, works with fullScreen: false true
[x] number if center is false, set the X axis 0
[y] number if center is false, set the Y axis 0
[enableGrid] boolean Enable the grid pattern false
[gridSize] number Set the grid inner boxes size 10
[snapToGrid] boolean Enable snaping to grid false
[lineJoin] LineJoinEnum The default Line join LineJoinEnum.ROUND
[lineCap] LineCapEnum The default Line cap LineCapEnum.ROUND
[dasharray] string The default dash-array ''
[dashoffset] number The default dash-offset 0
